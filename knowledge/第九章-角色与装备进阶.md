# 第九章：角色与装备进阶

## 概述

第九章深入探讨 Lyra 项目中角色系统的进阶机制，涵盖换装系统、队伍系统、屏幕指示器、异步加载、荣誉与击杀记录、游戏阶段管理、机器人 AI 及外轮廓线效果。

## 知识点

### [换装系统](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/knowledge/ch9/ch9_01_换装系统.md)
- 角色部件（FLyraCharacterPart）的添加与移除
- FastArray（FLyraCharacterPartList）的网络同步
- 动画层选择（FLyraAnimLayerSelectionEntry）
- Controller 端与 Pawn 端的双组件协作
- 换装作弊与开发者设置

### [队伍系统](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/knowledge/ch9/ch9_02_队伍系统.md)
- 队伍接口 ILyraTeamAgentInterface（扩展 IGenericTeamAgentInterface）
- 队伍显示资产 ULyraTeamDisplayAsset
- 队伍信息层级：ALyraTeamInfoBase / PublicInfo / PrivateInfo
- 队伍子系统 ULyraTeamSubsystem
- 队伍创建组件 ULyraTeamCreationComponent（Experience 驱动）
- 玩家生成 ULyraPlayerSpawningManagerComponent
- TDM 模式的战场生成策略（UTDM_PlayerSpawningManagmentComponent）
- 异步观察队伍变化（UAsyncAction_ObserveTeam / ObserveTeamColors）

### [指示器系统](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/knowledge/ch9/ch9_03_指示器系统.md)
- 指示器描述符 UIndicatorDescriptor（数据/投影/布局）
- 指示器管理器 ULyraIndicatorManagerComponent
- 控件接口 IIndicatorWidgetInterface
- UMG 层 UIndicatorLayer → Slate 画布 SActorCanvas
- 5 种投影模式（ComponentPoint / BoundingBox / ScreenBoundingBox / ActorBoundingBox / ActorScreenBoundingBox）
- 吸附边缘与箭头指示
- 对象池复用
- 交互系统集成

### [AsyncMixin 异步加载](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/knowledge/ch9/ch9_04_AsyncMixin异步加载.md)
- FAsyncMixin 的非拷贝设计
- 链式加载序列（AsyncLoad → AsyncCondition → AsyncEvent）
- FLoadingState 步骤管理
- FAsyncCondition ticker 轮询（0.16s）
- 零内存开销：仅在使用时分配
- FAsyncScope 独立使用封装

### [荣誉与击杀记录](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/knowledge/ch9/ch9_05_荣誉与击杀记录.md)
- 荣誉定义（FLyraAccoladeDefinitionRow, DataRegistry）
- 通知消息（FLyraNotificationMessage）
- 荣誉播报 Widget（ULyraAccoladeHostWidget, FAsyncMixin）
- VerbMessage 系统（FLyraVerbMessage, UGameplayMessageSubsystem）
- 击杀消息流程：死亡 → Eliminate.Message → 多个处理器
- UAssistProcessor：伤害记录 + 助攻广播
- UElimChainProcessor：连杀检测（时间窗口 4.5s）
- UElimStreakProcessor：连胜检测
- 客户端播报（ALyraPlayerState::ClientBroadcastMessage）
- 击杀 Feed UI（ListView）
- 得分处理（TeamInfo/PlayerState TagStack）

### [游戏阶段系统](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/knowledge/ch9/ch9_06_游戏阶段系统.md)
- 阶段能力 ULyraGamePhaseAbility（绑定 GameplayTag）
- 阶段子系统 ULyraGamePhaseSubsystem（WorldSubsystem）
- 阶段互斥逻辑（父阶段存活 vs 旁支替换 vs 同级替换）
- 阶段观察者（WhenPhaseStartsOrIsActive / WhenPhaseEnds）
- 暖场阶段（无敌 + 倒计时）
- 战斗阶段（重置 + 清理场景）
- 计分阶段（全局无敌 + PlayNextGame）
- 开发工具 ULyraDevelopmentStatics（PIE 跳过暖场）
- 积分面板 UI

### [机器人系统与 AI](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/knowledge/ch9/ch9_07_机器人系统与AI.md)

### [外轮廓线](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/knowledge/ch9/ch9_08_外轮廓线.md)
