# 动画概念速查

## 概述

本节速查表列出 Lyra 动画系统中涉及的虚幻引擎专业动画概念（对应课程编号 077），并标注 UE 官方文档作为深入阅读参考。

## 概念列表

| 概念 | 说明 | 官方文档参考 |
|------|------|--------------|
| 动画节点函数 | Animation Blueprint 中的节点函数，用于封装动画逻辑 | 引擎文档 > 动画节点 |
| 属性访问 | 在动画蓝图中访问角色属性的方式 | 引擎文档 > 动画蓝图属性 |
| 混合节点 | Blend Nodes，用于混合多个动画姿势 | 引擎文档 > 动画混合 |
| 姿势扭曲 | Pose Warping，动态调整骨骼姿态以适应目标 | 引擎文档 > 姿势扭曲 |
| 距离匹配 | Distance Matching，根据移动距离匹配动画播放进度 | 引擎文档 > 距离匹配 |
| 手部 IK | Hand IK，手部与物体/武器的交互定位 | 引擎文档 > IK |
| Two Bone IK | 双骨骼 IK，三关节链的逆运动学解算 | 引擎文档 > Two Bone IK |
| 复制骨骼 | Copy Bone，将骨骼变换从一个骨骼复制到另一个 | 引擎文档 > 复制骨骼节点 |
| Transform Bone | 变换骨骼，直接设置骨骼的位置/旋转/缩放 | 引擎文档 > 变换骨骼节点 |
| 虚拟骨骼 | Virtual Bone，在动画蓝图中创建虚拟参考点 | 引擎文档 > 虚拟骨骼 |

## 动画蓝图节点分类

| 分类 | 常见节点 |
|------|---------|
| 混合 | Blend Poses By Bool, Blend Poses By Int, Layered blend per bone, Blend Space |
| 骨骼控制 | Two Bone IK, Hand IK, Copy Bone, Transform Bone, Fabrik |
| 姿态 | Pose Warping, Distance Matching |
| 工具 | Virtual Bone, Linked Anim Graph, State Machine |

## 使用建议

上述概念在动画层蓝图（ABP_ItemAnimLayersBase）和主动画蓝图（ABP_Mannequin_Base）中均有应用。建议在编辑器中结合动画蓝图的实际节点配置进行理解。
