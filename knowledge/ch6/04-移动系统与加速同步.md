# 移动系统与加速同步

## ULyraCharacterMovementComponent

`ULyraCharacterMovementComponent` 继承自 `UCharacterMovementComponent`，由 `ALyraCharacter` 通过 `SetDefaultSubobjectClass` 在构造函数中注入。

## 加速度复制

### 数据结构

```cpp
struct FLyraReplicatedAcceleration {
    uint8 AccelXYRadians;   // 水平方向角 [0, 255]，由 Cartesian→Polar 量化
    uint8 AccelXYMagnitude; // 水平大小 [0, 255]
    int8  AccelZ;           // 垂直分量 [-127, 127]
};
```

### 复制管线

```
服务端
  ALyraCharacter::PreReplication()
    → CartesianToPolar(FVector) → 量化 → FLyraReplicatedAcceleration

复制条件
  DOREPLIFETIME_CONDITION(ReplicatedAcceleration, COND_SimulatedOnly)

客户端
  ALyraCharacter::OnRep_ReplicatedAcceleration()
    → PolarToCartesian → FVector
    → ULyraCharacterMovementComponent::SetReplicatedAcceleration(FVector)
```

### SimulateMovement 覆盖

```cpp
void ULyraCharacterMovementComponent::SimulateMovement(float DeltaTime)
{
    // 如果开启了加速度复制，先应用复制的加速度
    if (bHasReplicatedAcceleration)
    {
        // 用复制的加速度驱动模拟移动
        Acceleration = ReplicatedAcceleration;
    }
    Super::SimulateMovement(DeltaTime);
}
```

## Engine 加速度复制机制

- CVar `p.EnableCharacterAccelerationReplication`：启用/禁用引擎级加速度复制
- `ACharacter::ShouldReplicateAcceleration()`：判断是否需要复制加速度
- `ACharacter::GatherCurrentMovement()`：收集当前加速度数据

## Tag 控制运动

`ULyraCharacterMovementComponent` 通过 GameplayTag 控制角色运动：

```cpp
FRotator ULyraCharacterMovementComponent::GetDeltaRotation(float DeltaTime) const
{
    if (HasMatchingGameplayTag(TAG_Gameplay_MovementStopped))
        return FRotator(0, 0, 0);  // 不旋转
    return Super::GetDeltaRotation(DeltaTime);
}

float ULyraCharacterMovementComponent::GetMaxSpeed() const
{
    if (HasMatchingGameplayTag(TAG_Gameplay_MovementStopped))
        return 0;  // 不动
    return Super::GetMaxSpeed();
}
```

## 地面检测

```cpp
struct FLyraCharacterGroundInfo {
    uint64 LastUpdateFrame;   // 缓存的帧计数
    FHitResult GroundHitResult; // 检测命中结果
    float GroundDistance;       // 离地距离
};
```

- `GetGroundInfo()` 按帧缓存，同一帧内重复调用不重复 trace
- MOVE_Walking 模式下 GroundDistance = 0
- 其他模式从胶囊中心减去半高向下做线检测，trace 距离 = `GroundTraceDistance + CapsuleHalfHeight`

## 移动模式打包 (PackNetworkMovementMode)

```cpp
namespace PackedMovementModeConstants {
    const uint32 GroundShift = FMath::CeilLogTwo(MOVE_MAX); // 3
    const uint8 CustomModeThr = 2 * (1 << GroundShift);     // 16
    const uint8 GroundMask = (1 << GroundShift) - 1;        // 7
}

uint8 PackNetworkMovementMode() const
{
    if (MovementMode != MOVE_Custom) {
        const uint8 GroundModeBit = (GroundMovementMode == MOVE_Walking ? 0 : 1);
        return uint8(MovementMode.GetValue()) | (GroundModeBit << GroundShift);
    } else {
        return CustomMovementMode + CustomModeThr;
    }
}
```

- 低 3 位：主移动模式（Walking=1, Falling=2, Swimming=3, Flying=4, Custom=6）
- 第 3 位：地面行走模式位（0=Walking, 1=NavWalking）
- 自定义模式偏移：`CustomMovementMode + 16`
