# 相机系统

## 概述

Lyra 的相机系统由相机组件、相机模式栈和相机管理器三层组成，支持多模式切换、混合过渡和 UI 相机叠加。

## 核心类体系

```
ALyraPlayerCameraManager (APlayerCameraManager)
  └── 管理整体相机行为、震屏、FOV

ULyraCameraComponent (UActorComponent) — CameraMode 宿主
  ├── ULyraCameraModeStack — 相机模式栈
  │     ├── ULyraCameraMode_ThirdPerson — 第三人称模式
  │     └── [其他 CameraMode] — 可扩展
  └── 混合/叠加/穿透处理

ULyraUICameraManagerComponent (UActorComponent) — UI 相机
  └── 处理 UI 叠层的独立相机
```

## ULyraCameraComponent

### 职责

- 持有 `ULyraCameraModeStack` 实例
- 提供 `GetCameraMode()`、`ChangeCameraMode()`、`AddCameraMode()` 等接口
- 每帧计算最终相机参数（位置、旋转、FOV）
- 处理相机穿透避免（调用 CameraMode 的穿透检测）
- 实现了 `ILyraCameraAssistInterface` 接口，供其他系统查询相机状态

### InitState 参与

CameraComponent 也是初始化状态机的参与者：

- `RegisterInitStateFeature()` 注册
- 等待 PawnExtension 到达 DataAvailable
- 在 DataAvailable → DataInitialized 阶段创建 CameraModeStack

### 关键方法

```cpp
void GetCameraView(float DeltaTime, FMinimalViewInfo& DesiredView);
void ChangeCameraMode(const FGameplayTag& ModeTag);
void AddCameraMode(const FGameplayTag& ModeTag, float BlendTime);
void ClearCameraMode();
```

## ULyraCameraMode

`ULyraCameraMode` 是相机模式的基类，定义了相机计算的标准接口。

### 关键属性

```cpp
// 相机模式 Tag（用于识别和查找）
FGameplayTag CameraModeTag;

// 视野
float FieldOfView;

// 后处理设置
FPostProcessSettings PostProcessBlendSettings;
float PostProcessBlendWeight;

// 混合
float BlendTime;        // 进入此模式的过渡时间
float BlendAlpha;       // 当前混合系数
EAlphaBlendOption BlendFunction; // 混合函数

// 穿透避免
FLyraPenetrationAvoidanceFeeler PenetrationAvoidanceFeeler;
```

### 核心方法

```cpp
virtual void UpdateCamera(float DeltaTime, FCameraModeView& OutView);
virtual void UpdateView(float DeltaTime, FCameraModeView& OutView);
virtual void UpdatePrecalcs(float DeltaTime);
void GetCameraModeView(FMinimalViewInfo& DesiredView, float DeltaTime);
```

### FCameraModeView

```cpp
struct FCameraModeView {
    FVector Location;
    FRotator Rotation;
    float FieldOfView;
};
```

## ULyraCameraModeStack

相机模式栈管理模式模式的入栈/出栈/混合。

### 栈操作

```cpp
void PushMode(TSubclassOf<ULyraCameraMode> ModeClass);   // 入栈
void PopMode(TSubclassOf<ULyraCameraMode> ModeClass);    // 出栈
void EvaluateStack(float DeltaTime, FCameraModeView& OutView);  // 逐层计算
void BlendStack(FCameraModeView& OutView);               // 混合所有层
```

### 混合策略

1. 从栈底到栈顶逐层计算每个 CameraMode 的 `FCameraModeView`
2. 使用混合系数（BlendAlpha）在相邻层之间平滑过渡
3. 支持多种混合函数（Linear、Cubic、HermiteCubic、Sinusoidal、QuadraticInOut、CubicInOut、HermiteCubicInOut、QuarticInOut、QuinticInOut、CircularIn、CircularOut、CircularInOut、ExpIn、ExpOut、ExpInOut、Custom）

### 层式叠加

CameraMode 栈的混合是各层独立计算结果后按权重叠加，而不是简单的替换。这使得多个相机效果可以共存（如瞄准放大 + 震屏相机）。

## ULyraPlayerCameraManager

继承自 `APlayerCameraManager`，是 PlayerController 的默认相机管理器。

### 关键覆盖

- 创建时初始化 CameraComponent
- 覆盖 `UpdateCamera` 使用 CameraComponent 的计算结果
- 处理相机震屏（CameraShake）
- 管理 FOV 和 PostProcess 设置

## ULyraUICameraManagerComponent

为 UI 提供独立的相机视角计算，防止 UI 相机被游戏相机覆盖。

- 独立于主相机系统运行
- 用于 UI 过渡动画、加载画面、菜单等场景

## ILyraCameraAssistInterface

```cpp
interface ILyraCameraAssistInterface {
    virtual void GetCameraView(float DeltaTime, FMinimalViewInfo& DesiredView) = 0;
    virtual bool IsFeathering() const { return false; } // 是否为插接羽化状态
};
```

- 由 ULyraCameraComponent 实现
- 供其他系统（AI、动画、Ability）查询当前相机状态

## 相机系统数据流

```
每帧更新:
  ALyraPlayerCameraManager::UpdateCamera
    → ULyraCameraComponent::GetCameraView
      → ULyraCameraModeStack::EvaluateStack
          → ULyraCameraMode::UpdateCamera (逐层)
              → ULyraCameraMode::UpdateView (子类实现)
              → 穿透检测 (PenetrationAvoidance)
          → ULyraCameraModeStack::BlendStack (混合所有层)
      → 返回 FMinimalViewInfo
    → APlyerCameraManager 应用最终视角
```
