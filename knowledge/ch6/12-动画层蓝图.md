# 动画层蓝图

## 概述

动画层蓝图（Animation Layer Blueprint）实现特定动画层的内部复杂逻辑，通过动画层接口（Layer Interface）与主动画蓝图连接。Lyra 使用 `ALI_ItemAnimLayers` 作为接口，`ABP_ItemAnimLayersBase` 作为实现。对应课程编号 076，具体编辑界面截图见 [原版 .docx 讲义](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/docs/UE5_Lyra学习指南_076_构建动画层蓝图.docx)。

## 架构层次

```
FullBody (全身层)
  ├── FullBody_IdleState — 空闲状态
  │     ├── 空闲轮播 (Idle Break)
  │     ├── 原地转身 (Turn In Place)
  │     └── 原地转身恢复
  ├── FullBody_StartState — 起步状态
  ├── FullBody_CycleState — 循环状态
  ├── FullBody_StopState — 停止状态
  ├── FullBody_PivotState — 回转状态
  ├── FullBody_JumpStartLoopState — 起跳循环
  ├── FullBody_JumpStartState — 起跳
  ├── FullBody_JumpApexState — 跳跃顶点
  ├── FullBody_FallLoopState — 坠落循环
  └── FullBody_FallLandState — 落地

LeftHandPose_OverrideState — 左手姿态覆盖

FullBody_SkeletalControls — 骨骼控制器 (IK)

FullBody_Aiming — 瞄准姿态

FullBodyAdditives — 附加动画层
```

## 子状态详解

### IdleState（空闲状态）

**空闲轮播（Idle Break）：**
- 建立轮播时间（倒计时机制）
- 过渡条件 `Idle → IdleBreak`：倒计时归零时触发
- 选择轮播动画：随机选取预设动画

**原地转身（Turn In Place）：**
- `SetupTurnInplaceAnim` — 重设动画推进时间
- `SetUpTurnInPlaceRotationState` — 获取转向方向
- `UpdateTurnInPlaceAnim` — 更新推进转向的动画

**原地转身恢复：**
- 过渡条件满足时进入恢复状态
- 恢复播放时间位置
- 最终进入 Idle 状态或再次触发旋转

### StartState（起步状态）

- `SetupStartAnim` — 选择起步动画，重设参数
- `UpdateStartAnim` — 推进动画播放时间，计算步幅 Alpha 值，根据距离选择动画播放时间
- `UpdateHipFireRaiseWeaponPose` — 混入腰射动画

> 此部分涉及姿势扭曲（Pose Warping）和距离匹配（Distance Matching）等技术

### CycleState（循环状态）

持续播放行走/跑步循环动画，由速度驱动。

### StopState（停止状态）

- `SetUpStopAnim` — 当前有速度但无加速度时，匹配速度为 0
- `UpdateStopAnim` — 推进停止动画

### PivotState（回转状态）

- `SetUpPivotAnim` — 建立回转动画选择，初始化参数
- `UpdatePivotAnim` — 根据回转进度先减速后加速，匹配不同的推进动画方式

### Jump/Fall 状态序列

```
JumpStartLoop → JumpStart → JumpApex → FallLoop → FallLand
```

各状态按跳跃/坠落的生命周期依次推进。

### SkeletalControls（骨骼控制器）

根据动画曲线值和 bool 值控制 IK 的混入大小。

## 动画层与主动画蓝图的协作

```
主动画蓝图状态机
  ↓ 状态变更
动画层蓝图 (ABP_ItemAnimLayersBase)
  ↓ 姿态计算
通过动画层接口 (ALI_ItemAnimLayers) 返回数据
  ↓
主动画蓝图应用最终姿态
```
