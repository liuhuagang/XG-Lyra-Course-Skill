# 动画系统概览

## 概述

Lyra 的动画系统围绕 Locomotion 框架搭建，包含物理资产、后处理动画、FootIK、动画蓝图层级、动画层接口等结构。本节对应课程编号 071，其中包含的 UI 操作界面截图由 [讲义原文](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/docs/UE5_Lyra学习指南_071_动画系统.docx) 提供。

> 注意：本课程讲师并非专业动画 TA，关于 Locomotion 动画系统的讲解仅供参考。

## 动画资产体系

```
骨骼网格体
  ├── LOD 资产（每平台独立配置）
  ├── 残影设置（Ghost/拖尾效果，骨骼网格体上开启）
  └── 物理资产
        └── 自定义物理材质（命中检测 Tag、弱点伤害）

后处理动画
  ├── ABP_Manny_PostProcess — 限制过大的变形
  ├── CR_Mannequin_Procedural — 骨骼摆动/旋转传递计算
  └── CR_Mannequin_FootPlant — FootIK

IK 系统
  ├── IK_Mannequin — IK Rig 资产
  └── IK Retargeter — IK 重定向资产

复制/重定向蓝图
  ├── ABP_Mannequin_CopyPose — 复制姿态
  └── ABP_Mannequin_Retarget — 重定向姿态

主动画蓝图
  ├── ABP_Mannequin_Base — 主 Locomotion 动画蓝图
  └── ABP_Mannequin_TopDown — 俯视角动画蓝图

动画层
  ├── ALI_ItemAnimLayers — 动画层接口
  └── ABP_ItemAnimLayersBase — 具体动画层逻辑

Locomotion 配置
  ├── 手枪/步枪/霰弹枪/空手 — 各武器配置
  └── 配置动画资产即可
```
