# 第三章 设置系统

## 概述

本章是 Lyra 设置系统的核心。设置系统采用双层架构：本地设置（ULyraSettingsLocal）继承自 UGameUserSettings，存储于本地配置文件，管理音频、视频、性能、帧率等与机器绑定的设置；共享设置（ULyraSettingsShared）继承自 ULocalPlayerSaveGame，存储于存档系统，管理语言、色盲、字幕、灵敏度等与用户绑定的设置。两层设置通过 ULyraGameSettingRegistry 统一注册到设置 UI。

## 源码目录

- `Settings/` — 本地设置（ULyraSettingsLocal）、共享设置（ULyraSettingsShared）、设置注册表（ULyraGameSettingRegistry）
- `Settings/CustomSettings/` — 自定义设置值类型（分辨率、画质、移动端 FPS、语言、音频输出设备、安全区、键盘映射）
- `Settings/Screens/` — 设置编辑器界面（安全区编辑器、亮度编辑器）
- `Settings/Widgets/` — 设置列表条目控件
- `Audio/` — 音频设置（ULyraAudioSettings）、音频混合子系统（ULyraAudioMixEffectsSubsystem）
- `Hotfix/` — 热修复管理器（ULyraHotfixManager）、文本热修复配置（LyraTextHotfixConfig）
- `GameFeatures/` — GameFeature 策略（ULyraGameFeaturePolicy）、各种 GameFeatureAction
- `Performance/` — 平台渲染设置（ULyraPlatformSpecificRenderingSettings）、性能设置
- `AbilitySystem/` — GameplayCue 管理器（ULyraGameplayCueManager）

## 核心架构

### 双层设置架构

Lyra 将游戏设置拆分为两个层次：

1. **本地设置（ULyraSettingsLocal）**：继承自 UGameUserSettings，数据存储于本地配置文件（Config 标记）。与机器绑定，包括音频设备、视频分辨率、画质等级、帧率限制、控制器平台等。通过 `DefaultEngine.ini` 替换引擎默认的 UGameUserSettings。

2. **共享设置（ULyraSettingsShared）**：继承自 ULocalPlayerSaveGame，数据存储于存档系统。与用户绑定，跨平台同步，包括色盲模式、字幕选项、语言文化、手柄灵敏度、背景音频策略等。

### 设置注册表模式

ULyraGameSettingRegistry 作为设置注册中心，将本地设置和共享设置中的属性统一注册为 UGameSetting 对象，按类别分组（视频、音频、玩法、鼠标键盘、手柄），提供给设置 UI 使用。

### 设置生命周期

```
初始化阶段：
DefaultEngine.ini → UGameUserSettings 替换为 ULyraSettingsLocal
  → LoadSettings() 读取本地配置
  → ULyraGameSettingRegistry 注册所有设置项

运行时：
设置 UI 修改 → 写入属性 → MarkDirty
  → SaveChanges() → ApplySettings()
    → ApplyNonResolutionSettings() 应用非分辨率设置
    → ApplyResolutionSettings() 应用分辨率设置
  → SaveSettings() 持久化

共享设置：
AsyncLoadOrCreateSettings() 异步加载
  → ApplySettings() 应用（字幕、背景音频、文化、输入）
  → SaveSettings() 异步保存到存档
```

## 依赖关系

```
ULyraSettingsLocal (机器绑定，Config 序列化)
       │
       ├──→ ULyraAudioSettings (ControlBus 路径配置)
       │         └──→ ULyraAudioMixEffectsSubsystem (运行时加载 ControlBus)
       │
       ├──→ ULyraPlatformSpecificRenderingSettings (平台帧率模式)
       │
       └──→ ULyraGameSettingRegistry (注册所有设置项到 UI)
                  │
                  └──→ ULyraSettingsShared (用户绑定，存档序列化)

ULyraHotfixManager (运行时配置热修复)
       │
       └──→ 修改 ULyraSettingsLocal 等

ULyraGameFeaturePolicy (GameFeature 插件生命周期管理)
       │
       ├──→ UGameFeatureAction_* (具体动作)
       └──→ ULyraGameplayCueManager (GameplayCue 路径注册)
```
