# 底部操作栏

## 概述

底部操作栏（Bottom Action Bar）是 Lyra UI 中用于显示持久操作按钮的组件。它基于 CommonUI 的 `UCommonBoundActionBar` 体系，通过 UIActionBinding 机制自动收集和显示已注册的操作按钮。

## 核心类

### UCommonBoundActionBar（容器）

- **职责**：自动收集所有已注册的 UIActionBinding，并在操作栏中显示对应的按钮
- **子 Widget**：`W_BottomActionBar`（蓝图）

### UCommonBoundActionButton（按钮基类）

- **职责**：操作栏中单个操作按钮的基类
- **子 Widget**：`W_BoundActionButton`（蓝图）

### ULyraBoundActionButton

- **头文件**：[LyraBoundActionButton.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/UI/Common/LyraBoundActionButton.h)
- **继承链**：`UCommonBoundActionButton` → `ULyraBoundActionButton`
- **职责**：Lyra 的操作按钮实现，支持输入法样式切换

## 注册方式

操作按钮的注册有两种方式：

### 方式一：Class Defaults 勾选

在 Widget 的 Class Defaults 中勾选：
- **"Is Back Handler"**：标记为返回处理者
- **"Is Back Action Displayed"**：在操作栏中显示

### 方式二：C++ 代码注册

```cpp
RegisterUIActionBinding(FBindUIActionArgs(InputActionData, true, ...));
```

第二个参数 `true` 表示该操作应在操作栏中显示。

## 输入法样式切换

`ULyraBoundActionButton` 支持根据当前输入设备切换按钮样式：
- **KeyboardStyle**：键盘/鼠标输入时的样式
- **GamepadStyle**：手柄输入时的样式
- **TouchStyle**：触屏输入时的样式

通过 `HandleInputMethodChanged` 事件监听输入设备变化。
