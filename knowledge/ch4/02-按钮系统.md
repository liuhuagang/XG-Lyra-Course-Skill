# 按钮系统

## 概述

Lyra 的按钮体系以 `ULyraButtonBase` 为核心基类，提供统一的按钮外观和行为管理。在 UE5.4+ 中，按钮样式管理从蓝图迁移到 C++ 内部实现。

## 核心类

### ULyraButtonBase

- **头文件**：[LyraButtonBase.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/UI/Foundation/LyraButtonBase.h)
- **继承链**：`UCommonButtonBase` → `ULyraButtonBase`
- **职责**：项目通用按钮基类

### 按钮状态

- **Normal**：正常状态
- **Selected**：选中状态（用于切换按钮）
- **Disabled**：禁用状态

## UE5.4+ 按钮内部变化

在 UE5.4+ 中，`UCommonButtonBase` 的内部实现发生了重要变化：

- 按钮样式管理从蓝图（ButtonStyle 引用）迁移到 C++ 内部管理
- 通过 `SetButtonStyle()` 内部方法在 C++ 层面控制外观切换
- 按钮状态的切换现在完全由 C++ 驱动

## 按钮类型

- **普通按钮**：单次点击触发，无状态保持
- **切换按钮（Toggleable）**：支持按下/弹起两种状态的保持
- **动作按钮**：绑定到特定 UIAction 的按钮，如返回、应用、取消等
