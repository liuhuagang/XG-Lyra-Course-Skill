# 会话系统

## 概述

Lyra 的会话系统基于 `UCommonSessionSubsystem` 构建，支持创建、查找、加入游戏会话。系统同时支持 OSSv1 和 OSSv2（Online Subsystem），并提供 QuickPlay 机制实现快速匹配。

## 核心类

### UCommonSessionSubsystem

- **继承链**：`UGameInstanceSubsystem` → `UCommonSessionSubsystem`
- **职责**：管理在线会话的完整生命周期

### ULyraUserFacingExperienceDefinition

- **头文件**：[LyraUserFacingExperienceDefinition.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/GameModes/LyraUserFacingExperienceDefinition.h)
- **职责**：定义用户体验配置，创建 HostSessionRequest

### UCommonSession_HostSessionRequest

- **职责**：托管会话请求参数对象

### UCommonSession_SearchSessionRequest

- **职责**：查找会话请求参数对象

### UCommonSession_SearchResult

- **职责**：查找结果对象

## 会话流程

### 托管会话（HostSession）

```
HostSession(APlayerController*, UCommonSession_HostSessionRequest*)
  ├── Offline 模式：立即 ServerTravel(ConstructTravelURL())
  └── Online/LAN 模式：
        └── CreateOnlineSessionInternal(LocalPlayer, Request)
              ├── OSSv1: Sessions->CreateSession(*UserId, NAME_GameSession, HostSettings)
              │     HostSettings 包括:
              │       SETTING_GAMEMODE, SETTING_MAPNAME,
              │       SETTING_MATCHING_TIMEOUT=120,
              │       SETTING_SESSION_TEMPLATE_NAME="GameSession"
              └── OnCreateSessionComplete(FName SessionName, bool bWasSuccessful)
                    └── FinishSessionCreation(bWasSuccessful)
                          ├── 成功: ServerTravel(PendingTravelURL) + NotifyCreateSessionComplete
                          └── 失败: SetCreateSessionError + NotifyCreateSessionInformationUpdated(OutOfGame)
```

### 查找会话（FindSessions）

```
FindSessions(APlayerController*, UCommonSession_SearchSessionRequest*)
  └── FindSessionsInternalOSSv1:
        SearchSettings->QuerySettings.Set(SETTING_SESSION_TEMPLATE_NAME, "GameSession", Equals)
        Sessions->FindSessions(SearchSettings)
          └── OnFindSessionsComplete → 结果存入 SearchRequest->Results
```

### QuickPlay

```
QuickPlaySession(APlayerController*, UCommonSession_HostSessionRequest*)
  └── HandleQuickPlaySearchFinished
        ├── Results > 0: JoinSession(第一个结果)
        ├── Results == 0: HostSession
        └── 失败: NotifySessionInformationUpdated(OutOfGame)
```

### 加入会话（JoinSession）

```
JoinSession(APlayerController*, UCommonSession_SearchResult*)
  └── JoinSessionInternalOSSv1:
        Request->Result.Session.SessionSettings.bUsesPresence = true
        Sessions->JoinSession(*UserId, NAME_GameSession, Request->Result)
          └── 成功: ServerTravel
```

## ULyraUserFacingExperienceDefinition 配置

```cpp
CreateHostingRequest():
  Subsystem->CreateOnlineHostSessionRequest()
  设置: MapID, ModeNameForAdvertisement,
       ExtraArgs (Experience=FExperienceName, DemoRec),
       MaxPlayerCount=16
```

## UCommonSession_HostSessionRequest 参数

| 参数 | 类型 | 说明 |
|------|------|------|
| OnlineMode | ECommonSessionOnlineMode | Offline/LAN/Online |
| bUseLobbies | bool | 是否使用大厅 |
| bUseLobbiesVoiceChat | bool | 大厅语音聊天 |
| bUsePresence | bool | 使用 Presence |
| MapID | FPrimaryAssetId | 地图资源 ID |
| ModeNameForAdvertisement | FString | 广告模式名称 |
| ExtraArgs | TMap<FString, FString> | 额外参数（Experience、DemoRec 等） |
| MaxPlayerCount | int32 | 最大玩家数（默认 16） |

## ECommonSessionOnlineMode

- **Offline**：离线模式，直接 ServerTravel，不创建在线会话
- **LAN**：局域网模式
- **Online**：在线模式
