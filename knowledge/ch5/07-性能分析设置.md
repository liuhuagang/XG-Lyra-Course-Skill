# 性能分析设置

## 概述

性能分析设置是第五章中最复杂的子系统，涉及 CPU/GPU 性能数据采集、网络统计（Ping/丢包/包速率）、延迟追踪（NVIDIA Reflex），以及 Slate 自定义图表的实时渲染。

性能分析设置通过 `AddPerformanceStatPage` 函数创建，位于 `LyraGameSettingRegistry_PerfStats.cpp`。

## AddPerformanceStatPage 结构

```
PerformanceStatsPage (UGameSettingCollectionPage)
  ├── DisplayName: "Performance Stats"
  │
  ├── Performance (UGameSettingCollection)
  │     └── [按 ELyraDisplayablePerformanceStat 生成的设置项 × N]
  │           ├── ClientFPS → ULyraSettingValueDiscrete_PerfStat
  │           ├── ServerFPS → ULyraSettingValueDiscrete_PerfStat
  │           ├── IdleTime → ULyraSettingValueDiscrete_PerfStat
  │           ├── FrameTime → ULyraSettingValueDiscrete_PerfStat
  │           ├── FrameTime_GameTime → ULyraSettingValueDiscrete_PerfStat
  │           ├── FrameTime_RenderTime → ULyraSettingValueDiscrete_PerfStat
  │           ├── FrameTime_RHIThread → ULyraSettingValueDiscrete_PerfStat
  │           └── FrameTime_GPUTime → ULyraSettingValueDiscrete_PerfStat
  │
  ├── Network (UGameSettingCollection)
  │     ├── Ping → ULyraSettingValueDiscrete_PerfStat
  │     ├── PacketLoss_Incoming → ULyraSettingValueDiscrete_PerfStat
  │     ├── PacketLoss_Outgoing → ULyraSettingValueDiscrete_PerfStat
  │     ├── PacketRate_Incoming → ULyraSettingValueDiscrete_PerfStat
  │     ├── PacketRate_Outgoing → ULyraSettingValueDiscrete_PerfStat
  │     ├── PacketSize_Incoming → ULyraSettingValueDiscrete_PerfStat
  │     └── PacketSize_Outgoing → ULyraSettingValueDiscrete_PerfStat
  │
  └── Latency (UGameSettingCollection)
        ├── Latency_Total → ULyraSettingValueDiscrete_PerfStat
        │     └── 编辑条件: FGameSettingEditCondition_LatencyStatsSupported
        ├── Latency_Game → ULyraSettingValueDiscrete_PerfStat
        │     └── 编辑条件: FGameSettingEditCondition_LatencyStatsCurrentlyEnabled
        └── Latency_Render → ULyraSettingValueDiscrete_PerfStat
              └── 编辑条件: FGameSettingEditCondition_LatencyMarkersSupported
```

## ELyraDisplayablePerformanceStat

- **头文件**：[LyraPerformanceStatTypes.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Performance/LyraPerformanceStatTypes.h)

```cpp
enum class ELyraDisplayablePerformanceStat : uint8
{
    ClientFPS,              // 客户端帧率
    ServerFPS,              // 服务器帧率
    IdleTime,               // 空闲时间
    FrameTime,              // 帧时间（总）
    FrameTime_GameThread,   // 游戏线程帧时间
    FrameTime_RenderThread, // 渲染线程帧时间
    FrameTime_RHIThread,    // RHI 线程帧时间
    FrameTime_GPU,          // GPU 帧时间
    Ping,                   // 网络延迟
    PacketLoss_Incoming,    // 入站丢包率
    PacketLoss_Outgoing,    // 出站丢包率
    PacketRate_Incoming,    // 入站包速率
    PacketRate_Outgoing,    // 出站包速率
    PacketSize_Incoming,    // 入站包大小
    PacketSize_Outgoing,    // 出站包大小
    Latency_Total,          // 总延迟（端到端）
    Latency_Game,           // 游戏延迟
    Latency_Render,         // 渲染延迟

    Count                   // 静态检查 = 18
};
static_assert(ELyraDisplayablePerformanceStat::Count == 18, "...");
```

## ULyraPerformanceStatSubsystem

- **头文件**：[LyraPerformanceStatSubsystem.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Performance/LyraPerformanceStatSubsystem.h)
- **继承链**：`UObject` → `UGameInstanceSubsystem` → `ULyraPerformanceStatSubsystem`
- 是性能统计系统的中心协调者

```
ULyraPerformanceStatSubsystem
  ├── Tracker (FLyraPerformanceStatCache*)
  │     └── 性能数据缓存
  ├── StatDataTMap (TMap<ELyraDisplayablePerformanceStat, FSampledStatCache>)
  │     └── 每个指标的采样数据
  └── GetStatValue(Stat, bGetAverage) → double
        └── 返回值 = GetLastCachedStat(Stat) | GetAverage(Stat)
```

## FLyraPerformanceStatCache

- **继承链**：`IPerformanceDataConsumer`
- 通过 `GEngine->AddPerformanceDataConsumer(Tracker)` 注册到引擎管线

### ProcessFrame 数据处理

```
IPerformanceDataConsumer::FFrameData
  ├── DeltaSeconds
  ├── TrueDeltaSeconds
  ├── IdleSeconds
  ├── GameThreadTimeSeconds
  ├── RenderThreadTimeSeconds
  ├── RHIThreadTimeSeconds
  ├── GPUTimeSeconds
  └── ...

FLyraPerformanceStatCache::ProcessFrame(FrameData)
  ├── 计算 ClientFPS = 1.0 / FrameData.TrueDeltaSeconds
  ├── 计算 ServerFPS（从 UWorld::GetServerWorldTimeSecondsDelta）
  ├── 记录 IdleTime = FrameData.IdleSeconds
  ├── 记录 FrameTime = FrameData.TrueDeltaSeconds
  ├── 记录 GameThread/RenderThread/RHIThread/GPU 帧时间
  ├── Ping（从 PlayerState）
  ├── PacketLoss/Rate/Size（从 UNetConnection）
  └── Latency（从 ILatencyMarkerModule）
```

## FSampledStatCache

- 环形缓冲区实现，默认容量 125 个采样点
- 自动覆盖最旧数据

| 方法 | 说明 |
|------|------|
| `GetLastCachedStat()` | 获取最后缓存值 |
| `GetAverage()` | 获取平均值 |
| `GetMin()` | 获取最小值 |
| `GetMax()` | 获取最大值 |
| `ForEachCurrentSample(Func)` | 遍历所有当前样本 |

## Slate 图表渲染

### SLyraLatencyGraph

- **继承链**：`SLeafWidget`
- 自定义 `OnPaint` 实现

```
OnPaint(Args, OutDrawElements, Layer, ...)
  ├── 绘制背景
  ├── 绘制网格线
  ├── 绘制目标线（绿色阈值线）
  ├── 对每个 Stat:
  │     ├── 收集采样点 → FVector2D[] 
  │     └── FSlateDrawElement::MakeLines(OutDrawElements, Layer, PaintGeometry, Points)
  └── 绘制图例
```

### Widget 层级

```
ULyraPerfStatContainerBase (UCommonUserWidget)
  └── [ULyraPerfStatWidgetBase × N]
        ├── 文本模式: 直接显示数值
        └── 图表模式: 内嵌 SLyraLatencyGraph 或 ULyraPerfStatGraph
```

## 编辑条件

### FGameSettingEditCondition_LatencyStatsSupported

- 检查平台是否支持延迟统计
- 调用 `ULyraSettingsLocal::DoesPlatformSupportLatencyTrackingStats()`

### FGameSettingEditCondition_LatencyStatsCurrentlyEnabled

- 检查延迟统计当前是否启用
- 绑定到 `OnLatencyStatIndicatorSettingsChangedEvent`
- 监听延迟标记设置变更

### FGameSettingEditCondition_LatencyMarkersSupported

- 检查平台是否支持延迟标记（NVIDIA Reflex）
- 调用 `ULyraSettingsLocal::DoesPlatformSupportLatencyMarkers()`
