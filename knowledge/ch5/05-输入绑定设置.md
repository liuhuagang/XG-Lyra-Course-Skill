# 输入绑定设置

## 概述

输入绑定分为键盘鼠标设置和手柄设置两大部分。键盘鼠标设置由 `InitializeMouseAndKeyboardSettings` 创建，手柄设置由 `InitializeGamepadSettings` 创建。

## 键盘鼠标设置

```
MouseAndKeyboardSettings (UGameSettingCollection)
  ├── DisplayName: "Keyboard & Mouse"
  │
  ├── Sensitivity (UGameSettingCollection)
  │     ├── MouseSensitivityX (UGameSettingValueScalarDynamic)
  │     │     └── 数据源: GET_SHARED_SETTINGS_FUNCTION_PATH(GetMouseSensitivityX)
  │     │     └── 范围: 0~10, 步进 0.01
  │     ├── MouseSensitivityY (UGameSettingValueScalarDynamic)
  │     │     └── 数据源: GET_SHARED_SETTINGS_FUNCTION_PATH(GetMouseSensitivityY)
  │     └── TargetingMultiplier (UGameSettingValueScalarDynamic)
  │           └── 数据源: GET_SHARED_SETTINGS_FUNCTION_PATH(GetTargetingMultiplier)
  │
  ├── Invert (UGameSettingCollection)
  │     └── InvertVerticalAxis (UGameSettingValueDiscreteDynamic_Bool)
  │           └── 数据源: GET_SHARED_SETTINGS_FUNCTION_PATH(GetInvertVerticalAxis)
  │
  └── Key Bindings (UGameSettingCollection)
        └── [按配置动态生成 ULyraSettingKeyboardInput × N]
              └── 来自 UEnhancedPlayerMappableKeyProfile::PlayerMappedKeys
```

### 灵敏度数据

鼠标灵敏度存储在 `ULyraSettingsShared` 中（用户绑定），通过 `GET_SHARED_SETTINGS_FUNCTION_PATH` 绑定。

### ULyraSettingBasedScalar

- **头文件**：[LyraInputModifiers.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Input/LyraInputModifiers.h)
- **继承链**：`UObject` → `UInputModifier` → `ULyraSettingBasedScalar`
- 通过反射（`FindPropertyByName`）从 `ULyraSettingsShared` 中读取属性值作为灵敏度缩放系数
- 用于将游戏设置中的灵敏度值应用到 Enhanced Input 的输入处理管线中

## 手柄设置

```
GamepadSettings (UGameSettingCollection)
  ├── DisplayName: "Gamepad"
  │
  └── [GamepadSensitivity (UGameSettingValueDiscrete)
        └── 使用 ULyraAimSensitivityData 数据资产
              └── 映射 ELyraGamepadSensitivity → float
                    └── Slow / Slow2 / Normal / Fast2 / Fast / Insane...
              └── ULyraInputModifierGamepadSensitivity 读取
  ```

### ULyraAimSensitivityData

- **头文件**：[LyraAimSensitivityData.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Input/LyraAimSensitivityData.h)
- **继承链**：`UObject` → `UPrimaryDataAsset` → `ULyraAimSensitivityData`
- 将 `ELyraGamepadSensitivity` 枚举值映射为 `float` 灵敏度值

### ULyraInputModifierGamepadSensitivity

- **头文件**：[LyraInputModifiers.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Input/LyraInputModifiers.h)
- 从 `ULyraSettingsShared` 读取 `GetGamepadLookSensitivityPreset` 和 `GetGamepadTargetingSensitivityPreset`
- 结合 `ULyraAimSensitivityData` 数据资产计算出实际灵敏度值

### ULyraInputModifierAimInversion

- 从 `ULyraSettingsShared` 读取 `GetInvertVerticalAxis`/`GetInvertHorizontalAxis`
- 根据反转状态反转输入值

## 按键绑定系统

### 初始化流程

```
InitializeMouseAndKeyboardSettings
  └── UserSettings = ULyraLocalPlayer->GetUserSettings<ULyraInputUserSettings>()
        └── KeyProfiles = UserSettings->GetAllAvailableKeyProfiles()
              └── 对每个 Profile:
                    └── MappingRows = Profile->GetPlayerMappingRows()
                          └── 对每个 MappingRow:
                                └── 创建 ULyraSettingKeyboardInput
                                      ├── ActionMappingName = Row.MappingName
                                      ├── QueryOptions = FPlayerMappableKeyQueryOptions
                                      ├── ProfileIdentifier = Profile->GetProfileIdentifer()
                                      └── 添加到 Key Bindings 集合
```

### ULyraSettingKeyboardInput

- **头文件**：[LyraSettingKeyboardInput.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Settings/CustomSettings/LyraSettingKeyboardInput.h)
- **继承链**：`UObject` → `UGameSetting` → `UGameSettingValue` → `ULyraSettingKeyboardInput`

核心方法：

| 方法 | 说明 |
|------|------|
| `InitializeInputData(Profile, MappingData, QueryOptions)` | 初始化按键数据 |
| `ChangeBinding(Slot, NewKey)` | 更改按键绑定 |
| `GetAllMappedActionsFromKey(Key, QueryOptions)` | 获取按键已绑定的所有动作 |
| `IsMappingCustomized()` | 检查按键映射是否已被自定义 |

### 按键注册流程

```
UGameFeatureAction_AddInputContextMapping
  └── RegisterInputMappingContextsForLocalPlayer(LocalPlayer)
        └── Settings->RegisterInputMappingContext(IMC_InputConfiguration)
              └── Profile.PlayerMappedKeys.FindOrAdd(MappingName)
                    └── FPlayerMappedKey 包含 Slot 和 Key
```

### EPlayerMappableKeySlot

```
First / Second / Third / Fourth / Fifth / Sixth / Seventh / Unspecified / Max
```

输入设置界面默认使用 `First` 和 `Second` 两个按键槽。
