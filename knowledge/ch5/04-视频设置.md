# 视频设置

## 概述

视频设置由 `InitializeVideoSettings` 函数创建，位于 `LyraGameSettingRegistry_Video.cpp`。另有 `InitializeVideoSettings_FrameRates` 补充帧率限制选项。视频设置是最复杂的设置集合，包含显示、图形、画质、高级图形四大子分组。

## AutoSetQuality 联动

AutoSetQuality 是一个关键编辑依赖链。当用户点击"自动设置画质"时：

```
AutoSetQuality (UGameSettingAction)
  └── 执行动作:
        ├── RunAutoBenchmark → ULyraSettingsLocal::RunAutoBenchmark()
        └── ResetToMobileDeviceDefaults
              │
              └── AddEditDependency 影响:
                    ├── OverallQuality → 值更新后触发 EditConditionChanged
                    ├── ResolutionScale → 重新计算编辑状态
                    ├── GlobalIllumination
                    ├── Shadows
                    ├── AntiAliasing
                    ├── ViewDistance
                    ├── Textures
                    ├── Effects
                    ├── Reflections
                    └── PostProcessing
```

各渲染品质设置的编辑条件：

```
AutoSetQuality 是否正在运行 →
  ├── 是 → 禁用所有品质选项（防止手动修改与自动检测冲突）
  └── 否 → 检查 OverallQuality
        ├── Custom → 启用所有品质选项
        └── 非 Custom → 禁用（由预设统一控制）
```

## 帧率限制

`InitializeVideoSettings_FrameRates` 在 VideoCollection 中填充帧率限制选项。使用 `FGameSettingEditCondition_FramePacingMode` 根据平台类型（MobileStyle/ConsoleStyle/DesktopStyle）控制选项的可见性。
