# 第六章 角色与输入系统

## 概述

第六章涵盖 Lyra 的角色体系、移动系统、输入系统、相机系统和动画系统，对应课程编号 065~079。本章聚焦角色类层级、初始化状态机、输入绑定管道、移动同步、动画蓝图状态机与层架构、相机模式栈等核心架构。

## 子系统一览

| 子系统 | 核心类 | 职责 |
|--------|--------|------|
| 角色基类 | `ALyraCharacter`、`ALyraPawn` | 角色类层级，Team 接口，死亡流程，FastShared 复制 |
| PawnExtension 组件 | `ULyraPawnExtensionComponent` | 初始化状态机中枢，ASC 管理，PawnData 驱动 |
| HeroComponent | `ULyraHeroComponent` | 英雄输入绑定、ASC 初始化、相机模式代理 |
| 移动组件 | `ULyraCharacterMovementComponent` | 加速度复制、Tag 控制运动、地面检测 |
| 动画系统 | 见 [ch6/09~13](#动画系统补充知识文件) | 动画资产体系、状态机、线程安全逻辑、层蓝图 |
| 动画实例 | `ULyraAnimInstance` | 动画蓝图基类，GameplayTag→属性映射 |
| 输入系统 | `ULyraInputComponent`、`ULyraInputConfig` | Enhanced Input 集成、输入绑定管道 |
| 相机系统 | `ULyraCameraComponent`、`ULyraCameraMode`、`ULyraCameraModeStack` | 相机模式栈、第三人称相机、穿透处理 |

## 架构关系

```
ALyraCharacter (AModularCharacter)
  ├── ULyraPawnExtensionComponent — 初始化状态机中枢
  ├── ULyraHeroComponent — 输入绑定/ASC 初始化
  ├── ULyraHealthComponent — 生命值管理
  ├── ULyraCameraComponent — 相机模式栈宿主
  └── ULyraCharacterMovementComponent — 自定义移动

初始化状态机 (4 态链)
  Spawned → DataAvailable → DataInitialized → GameplayReady
    ↑ PawnExtension 驱动
    ↑ HeroComponent、CameraComponent 监听并参与

输入绑定管道
  ULyraPlayerController
    └── ULyraHeroComponent::OnSetupInputComponents
          └── ULyraInputComponent 创建
                └── ULyraInputConfig::BindAbilityActivation
                      └── Enhanced Input 映射到 GA

相机模式栈
  ULyraCameraComponent
    └── ULyraCameraModeStack
          ├── ULyraCameraMode_ThirdPerson (主模式)
          └── 其他 CameraMode (切换/混合)
```
