## 架构关系

```
ULyraSettingScreen (设置屏幕 Widget)
  └── ULyraGameSettingRegistry (设置注册器)
        ├── VideoSettings (UGameSettingCollection)
        │     ├── Display (WindowMode, Resolution)
        │     │     └── PerformanceStatsPage
        │     ├── Graphics (ColorBlindMode, Brightness, SafeZone)
        │     ├── GraphicsQuality (DeviceProfileSuffix, OverallQuality, ResolutionScale, GI, Shadows, AA, ...)
        │     └── AdvancedGraphics (VerticalSync)
        ├── AudioSettings (UGameSettingCollection)
        │     ├── Volume (Overall, Music, SoundEffects, Dialogue, VoiceChat)
        │     └── Sound (SubtitlesPage, AudioOutputDevice, BackgroundAudio, HeadphoneMode, HDRAudio)
        ├── GameplaySettings (UGameSettingCollection)
        │     ├── Language (ULyraSettingValueDiscrete_Language)
        │     └── Replays (RecordReplay, KeepReplayLimit)
        ├── MouseAndKeyboardSettings (UGameSettingCollection)
        │     ├── Sensitivity (MouseSensitivityX/Y, TargetingMultiplier)
        │     ├── Invert (InvertVerticalAxis)
        │     └── Key Bindings (ULyraSettingKeyboardInput × N)
        └── GamepadSettings (UGameSettingCollection)
              ├── GamepadSensitivity (ULyraGamepadSensitivity)
              └── Invert (InvertHorizontalAxis)

UGameSettingRegistry 基类
  ├── FOnSettingChanged / FOnSettingEditConditionChanged / FOnSettingNamedAction / FOnExecuteNavigation 事件
  ├── TopLevelSettings (根设置集合)
  ├── RegisteredSettings (所有已注册设置)
  ├── UGameSettingRegistryChangeTracker (变更追踪)
  └── Initialize → OnInitialize → RegisterSetting → RegisterInnerSettings(递归)
```

## 关键模式

- **数据源宏**：`GET_LOCAL_SETTINGS_FUNCTION_PATH` 和 `GET_SHARED_SETTINGS_FUNCTION_PATH` 用于将设置控件与 `ULyraSettingsLocal`/`ULyraSettingsShared` 的属性自动绑定，内部使用 `GET_FUNCTION_NAME_STRING_CHECKED` 实现编译期安全检查
- **异步初始化**：`UGameSettingValueScalarDynamic::Startup` 支持异步数据源加载，通过 `OnDataSourcesReady` 回调完成初始化
- **编辑条件链**：通过 `AddEditDependency` 将设置串联（如 AutoSetQuality → OverallQuality → 各渲染品质选项自动禁用），条件变化时自动重算 EditableState
- **平台特性标签**：`FWhenPlatformHasTrait` 通过 GameplayTag 控制设置项的可见性和可用性（窗口模式、音频设备切换、背景音频、回放等）
- **Enhanced Input 集成**：按键绑定设置通过 `UEnhancedInputUserSettings::RegisterInputMappingContext` 注册映射，使用 `UEnhancedPlayerMappableKeyProfile` 管理按键配置
- **性能统计管线**：Engine 接口 `IPerformanceDataConsumer` → `FLyraPerformanceStatCache` 帧数据采集 → `FSampledStatCache` 环形缓冲区 → `ULyraPerfStatWidgetBase` 显示 / `SLyraLatencyGraph` Slate 渲染
- **编译期安全检测**：`GET_FUNCTION_NAME_STRING_CHECKED` 宏通过 `sizeof(&ClassName::FunctionName)` 在编译期检测函数是否存在，零运行时开销
