## 架构层级

```
Item Definition Layer（定义层）
  ULyraInventoryItemDefinition —— 物品模板（UObject, Blueprintable, Abstract）
    └── Fragments[] —— 物品片段数组（EditInlineNew）
          ULyraInventoryItemFragment —— 片段基类
            ├── 自定义子类（定义物品行为/属性）

Item Instance Layer（运行时实例层）
  ULyraInventoryItemInstance —— 运行时物品实例
    ├── ItemDef —— 指向定义
    └── StatTags —— 统计标签堆栈（Replicated）

Inventory List Layer（清单层）
  FLyraInventoryList —— FFastArraySerializer
    └── Entries[] —— FLyraInventoryEntry
          ├── Instance -> ULyraInventoryItemInstance
          └── StackCount —— 堆叠数量

Inventory Manager Layer（管理器层）
  ULyraInventoryManagerComponent —— UActorComponent
    └── InventoryList —— 物品清单
```

## ULyraInventoryItemDefinition

[ULyraInventoryItemDefinition](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Inventory/LyraInventoryItemDefinition.h) 是抽象 Blueprintable UObject，作为物品模板。

| 成员 | 类型 | 说明 |
|------|------|------|
| `DisplayName` | FText | 物品显示名称 |
| `Fragments` | TArray\<TObjectPtr\<ULyraInventoryItemFragment\>\> | 物品片段（Instanced） |

**核心方法：**
- `FindFragmentByClass(TSubclassOf<ULyraInventoryItemFragment>)` —— 按类查找片段

## ULyraInventoryItemFragment

[ULyraInventoryItemFragment](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Inventory/LyraInventoryItemDefinition.h) 是 `EditInlineNew` 抽象基类，定义物品的特定行为。

| 方法 | 说明 |
|------|------|
| `OnInstanceCreated(ULyraInventoryItemInstance*)` | 实例创建时回调 |

在物品定义 BP 中添加片段子类来扩展物品功能（如武器段、消耗品段等）。

## ULyraInventoryItemInstance

[ULyraInventoryItemInstance](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Inventory/LyraInventoryItemInstance.h) 是运行时物品实例（Replicated）。

| 成员 | 类型 | 说明 |
|------|------|------|
| `StatTags` | FGameplayTagStackContainer | 统计标签堆栈（Replicated） |
| `ItemDef` | TSubclassOf\<ULyraInventoryItemDefinition\> | 物品定义（Replicated） |

**核心方法：**
- `AddStatTagStack(Tag, Count)` —— 增加统计标签计数
- `RemoveStatTagStack(Tag, Count)` —— 减少统计标签计数
- `GetStatTagStackCount(Tag)` —— 获取统计标签计数
- `HasStatTag(Tag)` —— 检查是否拥有某统计标签
- `GetItemDef()` —— 获取物品定义
- `FindFragmentByClass(Class)` —— 按类查找片段（委托给 ItemDef）

## FLyraInventoryList 与 FLyraInventoryEntry

[FLyraInventoryList](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Inventory/LyraInventoryManagerComponent.h) 继承 `FFastArraySerializer`，实现网络增量复制。

**FLyraInventoryEntry：**
| 成员 | 类型 | 说明 |
|------|------|------|
| `Instance` | TObjectPtr\<ULyraInventoryItemInstance\> | 物品实例 |
| `StackCount` | int32 | 堆叠数量 |
| `LastObservedCount` | int32 (NotReplicated) | 客户端上次观测计数 |

**FLyraInventoryList 方法：**
- `GetAllItems()` —— 获取全部物品
- `AddEntry(ItemClass, StackCount)` —— 按类添加
- `AddEntry(Instance)` —— 按实例添加
- `RemoveEntry(Instance)` —— 移除物品
- `NetDeltaSerialize` —— 网络增量序列化

## ULyraInventoryManagerComponent

[ULyraInventoryManagerComponent](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Inventory/LyraInventoryManagerComponent.h) 是主要 API 入口。

**核心方法：**

| 方法 | 说明 |
|------|------|
| `CanAddItemDefinition(ItemDef, StackCount)` | 检查能否添加物品 |
| `AddItemDefinition(ItemDef, StackCount)` | 按定义添加物品，返回实例 |
| `AddItemInstance(Instance)` | 直接添加实例 |
| `RemoveItemInstance(Instance)` | 移除实例 |
| `GetAllItems()` | 获取所有实例列表 |
| `FindFirstItemStackByDefinition(ItemDef)` | 按定义查找第一个堆叠 |
| `GetTotalItemCountByDefinition(ItemDef)` | 获取指定物品的总数量 |
| `ConsumeItemsByDefinition(ItemDef, NumToConsume)` | 消耗指定数量的物品 |

**FLyraInventoryChangeMessage：**
| 字段 | 类型 | 说明 |
|------|------|------|
| `InventoryOwner` | 拥有者组件 |
| `Instance` | 物品实例 |
| `NewCount` | 新数量 |
| `Delta` | 变动数量 |

## IPickupable 可拾取接口

[IPickupable](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Inventory/IPickupable.h) 接口定义可拾取物品的规范。

**FPickupTemplate：**
| 成员 | 说明 |
|------|------|
| `StackCount` | 堆叠数量 |
| `ItemDef` | 物品定义类 |

**FPickupInstance：**
| 成员 | 说明 |
|------|------|
| `Item` | 物品实例引用 |

**FInventoryPickup：**
| 成员 | 说明 |
|------|------|
| `Instances` | 实例数组 |
| `Templates` | 模板数组 |

**核心方法：**
- `GetPickupInventory()` —— 纯虚，返回 FInventoryPickup

**UPickupableStatics 蓝图函数库：**
- `GetFirstPickupableFromActor(Actor)` —— 从 Actor 获取 IPickupable
- `AddPickupToInventory(InventoryComponent, Pickup)` —— 将拾取物添加到背包（仅服务器）
