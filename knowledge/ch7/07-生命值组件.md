## ULyraHealthComponent

[ULyraHealthComponent](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Character/LyraHealthComponent.h) 是 UActorComponent，作为属性集（ULyraHealthSet）的 GameObject 侧桥梁，将属性变化转换为游戏事件。

## 职责边界

| 职责 | 归属 |
|------|------|
| 属性存储与网络同步 | ULyraHealthSet（数据层） |
| 属性变化的事件转译 | ULyraHealthComponent（逻辑层） |
| 死亡状态机管理 | ULyraHealthComponent |
| GameplayTag 清理 | ULyraHealthComponent |

## 死亡状态机

`ELyraDeathState` 枚举：

```
NotDead → DeathStarted → DeathFinished
```

| 状态 | 说明 |
|------|------|
| `NotDead` | 存活状态 |
| `DeathStarted` | 死亡开始（播放死亡动画/布娃娃等） |
| `DeathFinished` | 死亡完成（不可复活） |

**转换方法：**

```cpp
void StartDeath();   // NotDead → DeathStarted
void FinishDeath();  // DeathStarted → DeathFinished
```

**状态查询：**

- `GetDeathState()` —— 获取当前状态
- `IsDeadOrDying()` —— 是否已死或正在死亡（`DeathState != NotDead`）

## 属性事件委托

| 委托 | 参数 | 触发时机 |
|------|------|----------|
| `OnHealthChanged` | OldValue, NewValue, Instigator | Health 属性变化时 |
| `OnMaxHealthChanged` | OldValue, NewValue, Instigator | MaxHealth 属性变化时 |
| `OnDeathStarted` | OwningActor | 开始死亡时 |
| `OnDeathFinished` | OwningActor | 死亡完成时 |

## Handle 方法

当 HealthSet 的委托触发时，HealthComponent 的 Handle 方法被调用：

```cpp
void ULyraHealthComponent::HandleHealthChanged(AActor* Instigator, float OldValue, float NewValue)
{
    // 广播 OnHealthChanged 委托
    OnHealthChanged.Broadcast(this, OldValue, NewValue, Instigator);
}

void ULyraHealthComponent::HandleMaxHealthChanged(AActor* Instigator, float OldValue, float NewValue)
{
    OnMaxHealthChanged.Broadcast(this, OldValue, NewValue, Instigator);
}

void ULyraHealthComponent::HandleOutOfHealth(AActor* DamageInstigator, AActor* DamageCauser, ...)
{
    // 开始死亡序列
    StartDeath();
}
```

## 自毁

```cpp
void DamageSelfDestruct(bool bFellOutOfWorld);
```

当 `bFellOutOfWorld` 为 true 时，应用 `TAG_Gameplay_FellOutOfWorld` 标签的伤害；否则应用 `TAG_Gameplay_DamageSelfDestruct` 标签。

## GameplayTag 清理

```cpp
void ClearGameplayTags();
```

在死亡流程中调用，移除角色身上的特定 GameplayTag，确保死亡后状态正确。

## 调用架构

```
ULyraHealthSet::PostGameplayEffectExecute()
  └── OnHealthChanged.Broadcast(...)
        └── ULyraHealthComponent::HandleHealthChanged(...)
              └── OnHealthChanged.Broadcast(...)  // 对外委托

ULyraHealthSet::PostGameplayEffectExecute()
  └── OnOutOfHealth.Broadcast(...)
        └── ULyraHealthComponent::HandleOutOfHealth(...)
              └── StartDeath()
                    └── OnDeathStarted.Broadcast(...)
```
