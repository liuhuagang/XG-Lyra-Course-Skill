## IInteractableTarget（可交互目标接口）

[IInteractableTarget](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Interaction/IInteractableTarget.h) 定义可交互对象的核心接口。

```cpp
UINTERFACE()
class UInteractableTarget : public UInterface { };

class IInteractableTarget
{
    // 收集交互选项，由交互目标实现
    virtual void GatherInteractionOptions(
        const FInteractionQuery& InteractQuery,
        FInteractionOptionBuilder& OptionBuilder) = 0;

    // 定制交互事件数据，可选覆写
    virtual void CustomizeInteractionEventData(
        const FGameplayTag& InteractionEventTag,
        FGameplayEventData& InOutEventData) { }
};
```

## FInteractionOption

[FInteractionOption](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Interaction/InteractionOption.h) 描述一个交互选项：

| 字段 | 类型 | 说明 |
|------|------|------|
| `InteractableTarget` | TScriptInterface\<IInteractableTarget\> | 交互目标引用 |
| `Text` | FText | 交互显示文本 |
| `SubText` | FText | 交互副文本 |
| `InteractionAbilityToGrant` | TSubclassOf\<UGameplayAbility\> | 授予发起方的交互 GA |
| `TargetAbilitySystem` | TObjectPtr\<UAbilitySystemComponent\> | 目标 ASC |
| `TargetInteractionAbilityHandle` | FGameplayAbilitySpecHandle | 目标身上的交互 Ability |
| `InteractionWidgetClass` | TSoftClassPtr\<UUserWidget\> | 交互 UI Widget |

交互方式有两种：
1. **授予发起方 GA**：通过 `InteractionAbilityToGrant` 授予发起方一个交互 GA，GA 执行交互逻辑
2. **触发目标侧 Ability**：通过 `TargetAbilitySystem` + `TargetInteractionAbilityHandle` 触发目标身上的交互 Ability

## FInteractionOptionBuilder

非 UObject 的辅助构建器类：

```cpp
class FInteractionOptionBuilder
{
    TScriptInterface<IInteractableTarget> Scope;

    void AddInteractionOption(const FInteractionOption& Option)
    {
        // 自动将 Scope 赋予 Option.InteractableTarget
        Option.InteractableTarget = Scope;
    }
};
```

## ALyraWorldCollectable（世界可拾取物）

[ALyraWorldCollectable](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Plugins/GameFeatures/ShooterCore/Source/ShooterCoreRuntime/Public/LyraWorldCollectable.h) 同时实现 `IInteractableTarget` 和 `IPickupable`，是世界中的交互拾取物。

**继承链：** `AActor` + `IInteractableTarget` + `IPickupable`

| 成员 | 类型 | 说明 |
|------|------|------|
| `Option` | FInteractionOption | 交互选项配置 |
| `StaticInventory` | FInventoryPickup | 静态拾取物清单 |

**核心方法：**
- `GatherInteractionOptions(InteractQuery, InteractionBuilder)` —— 将自身作为交互选项添加
- `GetPickupInventory()` —— 返回拾取物清单

### 交互拾取完整流程

```
1. 玩家靠近 ALyraWorldCollectable
2. 检测与收集交互选项
     ├── 使用 Overlap/Hit 检测找到可交互 Actor
     ├── 调用 GetInteractableTargetsFromActor()
     │     └── 获取所有 IInteractableTarget 接口
     ├── 调用 GatherInteractionOptions()
     │     └── 填充 FInteractionOption 列表
     └── IInteractionInstigator::ChooseBestInteractionOption()
           └── 选出最佳交互选项

3. 执行交互
     ├── 授予交互 GA（InteractionAbilityToGrant）
     └── 交互 GA 激活
           ├── 调用 UPickupableStatics::AddPickupToInventory()
           │     └── 将 FInventoryPickup 写入背包
           │           └── ULyraInventoryManagerComponent::AddItemDefinition/AddItemInstance()
           └── 播放拾取效果/音效
                 └── 移除世界拾取物
```
