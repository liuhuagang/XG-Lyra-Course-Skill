# 远程武器定义与扩散系统

## 概述

远程武器系统在装备系统基础上扩展，定义射击武器的运行时数据，包括扩散（Spread）/热量（Heat）系统、多状态精度乘数、距离伤害衰减和材质伤害倍率。

```
  └─ ULyraWeaponInstance (ULyraEquipmentInstance)
       └─ ULyraRangedWeaponInstance (ULyraWeaponInstance, ILyraAbilitySourceInterface)
```

### 扩散/热量系统（Spread/Heat）

核心概念：**Heat（热量）** 作为中间值，通过曲线映射到 **SpreadAngle（扩散角）**。

```
每发子弹 AddSpread() → Heat↑ → HeatToSpreadCurve → SpreadAngle↑
时间推移 Tick() → UpdateSpread() → Heat↓（冷却） → SpreadAngle↓
```

#### 曲线定义

| 曲线 | X 轴 | Y 轴 | 作用 |
|------|------|------|------|
| HeatToSpreadCurve | Heat 范围 | Spread 角度 | Heat → 扩散角 |
| HeatToHeatPerShotCurve | 当前 Heat | 单发热量增量 | 每发增加的热量 |
| HeatToCoolDownPerSecondCurve | 当前 Heat | 冷却速率/秒 | 每秒散热的速率 |

#### 精度乘数系统（Multipliers）

四种状态乘数，每个独立插值过渡，最终合并为组合乘数：

| 乘数 | 检测方式 | 过渡速率 |
|------|----------|----------|
| StandingStillMultiplier | 速度 ≤ StandingStillSpeedThreshold（80cm/s） | TransitionRate_StandingStill |
| CrouchingMultiplier | bIsCrouched | TransitionRate_Crouching |
| JumpFallMultiplier | IsFalling | TransitionRate_JumpingOrFalling |
| AimingMultiplier | CameraComponent->GetBlendInfo() 检查 TAG_Lyra_Weapon_SteadyAimingCamera | 直接 |

**合并方式：** `CombinedMultiplier = AimingMultiplier * StandingStillMultiplier * CrouchingMultiplier * JumpFallMultiplier`

#### 首发射击精度（First Shot Accuracy）

条件：`bAllowFirstShotAccuracy && bMinMultipliers && bMinSpread`
- `bMinMultipliers` — 所有乘数都已过渡到最小值
- `bMinSpread` — 热量已冷却到最小值
- 生效时 `GetCalculatedSpreadAngleMultiplier()` 返回 0.0，即绝对精准

### Tick() 更新逻辑

```
Tick(DeltaSeconds):
    UpdateSpread(DeltaSeconds)       → 返回 bMinSpread
    UpdateMultipliers(DeltaSeconds)  → 返回 bMinMultipliers
    bHasFirstShotAccuracy = bAllowFirstShotAccuracy && bMinMultipliers && bMinSpread
```

### UpdateSpread() 流程

1. 计算 `TimeSinceFired = 当前时间 - LastFireTime`
2. 如果 `TimeSinceFired > SpreadRecoveryCooldownDelay`：
   - 从 `HeatToCoolDownPerSecondCurve` 获取冷却速率
   - `CurrentHeat = ClampHeat(CurrentHeat - CooldownRate * DeltaSeconds)`
3. `CurrentSpreadAngle = HeatToSpreadCurve.Eval(CurrentHeat)`
4. 返回 `CurrentSpreadAngle` 是否等于最小扩散角

### AddSpread() 流程（每发子弹触发）

1. 从 `HeatToHeatPerShotCurve.Eval(CurrentHeat)` 获取热量增量
2. `CurrentHeat = ClampHeat(CurrentHeat + HeatPerShot)`
3. `CurrentSpreadAngle = HeatToSpreadCurve.Eval(CurrentHeat)`

### UpdateMultipliers() 流程

1. **StandingStillMultiplier**：计算 `MovementTargetValue`（MappedRangeClamped 映射速度范围），通过 `FInterpTo` 过渡
2. **CrouchingMultiplier**：根据 `bIsCrouched` 在 1.0 和配置值之间通过 `FInterpTo` 过渡
3. **JumpFallMultiplier**：根据 `IsFalling` 在 1.0 和配置值之间通过 `FInterpTo` 过渡
4. **AimingAlpha**：通过 `GetPawn()->GetComponentByClass<ULyraCameraComponent>()->GetBlendInfo()` 检查是否混合了 `TAG_Lyra_Weapon_SteadyAimingCamera`
5. **Combined = AimingMultiplier × StandingStillMultiplier × CrouchingMultiplier × JumpFallMultiplier**

### 其他配置

- `BulletsPerCartridge` — 每发子弹数（霰弹枪 > 1）
- `MaxDamageRange` — 最大伤害距离（默认 25000cm）
- `BulletTraceSweepRadius` — 子弹追踪扫描半径（0.0 表示线追踪）
- `SpreadExponent` — 扩散指数（>1 时弹孔更聚集在中心）

### OnEquipped() 初始化

- `CurrentHeat` 设为热量范围中点
- `StandingStillMultiplier`、`JumpFallMultiplier`、`CrouchingMultiplier` 均初始化为 1.0
- 调用父类 OnEquipped
