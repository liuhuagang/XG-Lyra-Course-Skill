# 装备系统架构

## 概述

装备系统是 Lyra 中将物品/武器挂载到 Pawn 上的核心框架。采用三段式设计：**Definition（定义）→ Instance（实例）→ ManagerComponent（管理器）**，配合 QuickBar（快捷栏）实现玩家换装操作。

## 核心类层次

```
ULyraEquipmentDefinition (UObject, Const, Abstract)
  └─ 定义"如何装备"——实例类型、要授予的 AbilitySet、要生成的 Actor

ULyraEquipmentInstance (UObject, BlueprintType, 支持网络复制)
  └─ 运行时装备实例，持有 Instigator 和 SpawnedActors

ULyraEquipmentManagerComponent (UPawnComponent)
  └─ Pawn 级装备管理器，维护 FLyraEquipmentList

ULyraQuickBarComponent (UControllerComponent)
  └─ 快捷栏系统，管理 Inventory → Equipment 的装备流程
```

## 装备实例（ULyraEquipmentInstance）

[LyraEquipmentInstance.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Equipment/LyraEquipmentInstance.h)

```cpp
UCLASS(BlueprintType, Blueprintable)
class ULyraEquipmentInstance : public UObject
{
    UObject* Instigator;              // Replicated—通常是 InventoryItemInstance
    TArray<AActor*> SpawnedActors;    // Replicated—生成的 Actor

    void SpawnEquipmentActors(const TArray<FLyraEquipmentActorToSpawn>& ActorsToSpawn);
    void DestroyEquipmentActors();

    virtual void OnEquipped();        // 调用 K2_OnEquipped BP 事件
    virtual void OnUnequipped();      // 调用 K2_OnUnequipped BP 事件
};
```

- `GetPawn()` 通过 `Instigator` → `GetTypedOuter<APawn>()` 获取
- `IsSupportedForNetworking()` 返回 true，支持网络复制
- `OnEquipped()` 先调用 SpawnEquipmentActors 再触发 K2 事件
- `OnUnequipped()` 先调用 DestroyEquipmentActors 再触发 K2 事件

### FLyraEquipmentList（FastArraySerializer）

以 `FFastArraySerializer` 机制实现网络复制，包含 `TArray<FLyraAppliedEquipmentEntry> Entries`。

```cpp
struct FLyraAppliedEquipmentEntry : public FFastArraySerializerItem
{
    TSubclassOf<ULyraEquipmentDefinition> EquipmentDefinition;
    ULyraEquipmentInstance* Instance;
    FLyraAbilitySet_GrantedHandles GrantedHandles;  // Authority-only，不复制
};
```

**`AddEntry()` 流程：**
1. Spawn 传入 EquipmentDefinition 中定义的 `InstanceType`
2. 调用 `OnEquipped()` → 内部 SpawnEquipmentActors + K2_OnEquipped
3. 遍历 `AbilitySetsToGrant`，调用 `GiveToAbilitySystem(ASC, &GrantedHandles, Result)`

  > 此处 `Result` 是新创建的 EquipmentInstance，作为 `SourceObject` 传入 AbilitySet 授予流程，使得 Ability 可通过 `GetSourceObject()` 回溯到 EquipmentInstance

4. 将 Entry 添加到 `Entries` 数组

**`RemoveEntry()` 流程：**
1. 调用 `Instance->OnUnequipped()` → DestroyEquipmentActors + K2_OnUnequipped
2. `TakeFromAbilitySystem(GrantedHandles)` 收回所有授予的 Ability
3. 从 Entries 中移除

## 快捷栏（ULyraQuickBarComponent）

[查看源码](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Equipment/)

组件挂载在 Controller 上，管理 Inventory → Equipment 的装备触发。

### 核心数据

- `Slots: TArray<TObjectPtr<ULyraInventoryItemInstance>>` — Replicated，快捷栏槽位
- `ActiveSlotIndex: int32` — Replicated，当前激活槽位索引

### 核心流程

**装备物品到槽位** `AddItemToSlot(ULyraInventoryItemInstance* ItemInstance)`：
- 调用 `GetNextFreeItemSlot()` 找到空槽位
- 将 ItemInstance 放入该槽位

**激活槽位** `SetActiveSlotIndex(int32 NewIndex)`（Server Only）：
1. 如果当前槽位有装备 → `UnequipItemInSlot()`
2. 设置 `ActiveSlotIndex = NewIndex`
3. 如果新槽位有物品 → `EquipItemInSlot()`

**槽位装备** `EquipItemInSlot(int32 SlotIndex)`：
1. 从 `Slots[SlotIndex]` 获取 `ULyraInventoryItemInstance`
2. 查找该物品上的 `ULyraInventoryItemFragment_EquippableItem` Fragment
3. 从 Fragment 中获取 `EquipmentDefinition`
4. 获取 `EquipmentManagerComponent`
5. 调用 `EquipmentManager->EquipItem(EquipmentDefinition)` → 返回 `ULyraEquipmentInstance`
6. 调用 `EquippedItem->SetInstigator(SlotItem)` — 将 InventoryItemInstance 设为 Instigator

  > 关键设计：Instigator 指向 InventoryItemInstance，使得 Equipment 能回溯到 Inventory 中的物品

**循环切换** `CycleActiveSlotForward/Backward()`：
- 使用 `do-while` 循环包装索引
- 跳过空槽位
- 调用 `SetActiveSlotIndex`

**完整 Equip 链路：**

```
Inventory AddItemDefinition
  → QuickBar AddItemToSlot
    → QuickBar SetActiveSlotIndex
      → QuickBar EquipItemInSlot
        → EquipmentManager EquipItem
          → FLyraEquipmentList::AddEntry
            → Spawn EquipmentInstance
            → AbilitySet->GiveToAbilitySystem(ASC, GrantedHandles, Instance as SourceObject)
            → Spawn Equipment Actors
```

## 拾取定义

### ULyraPickupDefinition

DataAsset，定义拾取物属性：
- `InventoryItemDefinition` — 拾取后添加的物品定义
- `DisplayMesh` — 显示网格
- `SpawnCoolDownSeconds` — 重生冷却
- 拾取/重生音效和特效

### ULyraWeaponPickupDefinition

继承 ULyraPickupDefinition，增加武器网格偏移：
- `WeaponMeshOffset` — 武器网格位置偏移
- `WeaponMeshScale` — 武器网格缩放
