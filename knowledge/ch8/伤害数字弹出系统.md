# 伤害数字弹出系统（NumberPop）

## 概述

NumberPop 系统在命中目标时在屏幕空间或世界空间弹出伤害数字，提供视觉伤害反馈。支持 Niagara 文本和 Mesh 文本两种渲染方式。

## 类层次

```
LyraDamagePopStyle (UDataAsset)
  └─ LyraDamagePopStyleNiagara (UDataAsset)
       └─ Niagara 粒子系统的伤害数字样式

LyraNumberPopComponent (UActorComponent) [Abstract]
  ├─ LyraNumberPopComponent_NiagaraText
  └─ LyraNumberPopComponent_MeshText
```

## 伤害数字样式

### LyraDamagePopStyle

UDataAsset，定义伤害数字弹出样式的基础类。

### LyraDamagePopStyleNiagara

UDataAsset，使用 Niagara 粒子系统的伤害数字样式：
- 定义 Niagra 系统模板
- 数字渲染参数（颜色、大小、字体）
- 动画参数（浮动、淡出等）

## 数字弹出组件

### LyraNumberPopComponent

抽象基类，定义数字弹出示接口。

### LyraNumberPopComponent_NiagaraText

使用 Niagara 粒子系统渲染伤害数字：
- 基于 Niagra 的文本渲染
- 支持数字动画（上升、淡出）
- 可定制颜色和样式

### LyraNumberPopComponent_MeshText

使用静态网格体（3D Text）在世界空间渲染伤害数字：
- 适合在世界空间中定位的伤害数字
- 支持旋转朝向摄像机

## 触发流程

1. 伤害 GE 应用后，通过 GameplayCue 触发
2. GameplayCue 查找 Actor 上的 NumberPopComponent
3. 调用 NumberPopComponent 的显示接口
4. 根据伤害类型选择对应的 DamagePopStyle
5. 在命中位置弹出伤害数字
