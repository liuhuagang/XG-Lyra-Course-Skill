# 武器生成器与拾取系统

## 概述

武器生成器（Weapon Spawner）是在场景中生成可拾取武器的 Actor，玩家碰撞后拾取武器加入背包。

UCLASS(Blueprintable, BlueprintType)
class ALyraWeaponSpawner : public AActor
{
    ULyraWeaponPickupDefinition* WeaponDefinition;  // 武器拾取定义

    bool bIsWeaponAvailable;          // Replicated，武器是否可用
    float CoolDownTime;               // 拾取后冷却时间
    float CheckExistingOverlapDelay;  // 武器可用后延迟检查重叠

    UCapsuleComponent* CollisionVolume;      // 碰撞体积
    UStaticMeshComponent* PadMesh;           // 底座网格
    UStaticMeshComponent* WeaponMesh;        // 武器显示网格
    float WeaponMeshRotationSpeed;            // 武器旋转速度
};

## 核心流程

### 拾取流程（OnOverlapBegin）

1. 玩家进入碰撞体触发 `OnOverlapBegin`
2. 检查 `bIsWeaponAvailable` — 只有可用时才能拾取
3. 调用 `AttemptPickUpWeapon(Pawn)`
4. AttemptPickUpWeapon 调用 `GiveWeapon(WeaponItemClass, Pawn)`（BlueprintNativeEvent）
5. `GiveWeapon` 蓝图实现：创建 InventoryItemInstance 并添加到玩家背包
6. 拾取成功后调用 `StartCoolDown()` 开始冷却

### 冷却流程（StartCoolDown）

1. 调用 `SetWeaponPickupVisibility(false)` — 隐藏武器
2. 设置 `bIsWeaponAvailable = false`
3. 启动冷却定时器（CoolDownTime 后调用 OnCoolDownTimerComplete）
4. 通过 `CoolDownPercentage` 驱动冷却进度指示

### 重生流程

1. 冷却定时器触发 `OnCoolDownTimerComplete`
2. 设置 `bIsWeaponAvailable = true`
3. 调用 `SetWeaponPickupVisibility(true)` — 显示武器
4. 调用 `PlayRespawnEffects()` — 重生特效
5. 使用 `CheckExistingOverlapDelay` 延迟后调用 `CheckForExistingOverlaps()`
6. CheckForExistingOverlaps 检测底座上是否已有 Pawn 在等待拾取

### 网络复制

- `bIsWeaponAvailable` 使用 `ReplicatedUsing = OnRep_WeaponAvailability`
- OnRep_WeaponAvailability 处理客户端侧的可视状态和特效同步
- 冷却百分比通过 Tick 在客户端本地计算

## 拾取定义

### ULyraPickupDefinition

DataAsset 定义拾取物属性：
- `InventoryItemDefinition` — 拾取后添加的物品定义
- `DisplayMesh` — 拾取物显示网格
- `SpawnCoolDownSeconds` — 重生冷却时间
- 拾取和重生音效/特效

### ULyraWeaponPickupDefinition

继承 ULyraPickupDefinition，增加武器网格参数：
- `WeaponMeshOffset` — 武器网格位置偏移
- `WeaponMeshScale` — 武器网格缩放

## 蓝图关键函数

| 函数 | 类型 | 作用 |
|------|------|------|
| GiveWeapon | BlueprintImplementableEvent | 创建物品并添加到背包 |
| AttemptPickUpWeapon | BlueprintNativeEvent | 拾取触发逻辑 |
| PlayPickupEffects | BlueprintNativeEvent | 拾取音效/特效 |
| PlayRespawnEffects | BlueprintNativeEvent | 重生音效/特效 |
| GetDefaultStatFromItemDef | BlueprintCallable, BlueprintPure, Static | 从物品定义读取默认属性 |
