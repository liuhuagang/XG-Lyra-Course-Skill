# 指示器系统

## 架构概览

指示器系统在屏幕空间渲染世界物体的标记（血条、名称、交互提示等），支持吸附屏幕边缘、箭头指示、优先级排序和对象池复用。

## Slate 画布

[SActorCanvas](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/UI/IndicatorSystem/SActorCanvas.h)

```cpp
class SActorCanvas : public SPanel, public FAsyncMixin, public FGCObject
{
    // 子分类
    TArray<FSlot> CanvasChildren;   // 普通指示器
    TArray<FSlot> ArrowChildren;     // 吸附边缘箭头指示器
    FSlotChildren AllChildren;       // 合并子集

    // 每帧更新
    EActiveTimerReturnType UpdateCanvas(double CurrentTime, float DeltaTime);
    void UpdateActiveTimer();

    // 添加/移除指示器
    void AddIndicatorForEntry(UIndicatorDescriptor* Indicator);
    void RemoveIndicatorForEntry(UIndicatorDescriptor* Indicator);
};
```

### 核心特性

1. **预分配箭头**：构造时预分配 10 个箭头 Widget
2. **ActiveTimer**：每帧驱动投影计算和布局更新
3. **投影计算**：根据投影模式计算世界坐标到屏幕坐标的转换
4. **吸附边缘**：通过 `SegmentPlaneIntersection` 确定指示器在屏幕边缘的位置
5. **排序**：按 Priority → Depth 排序
6. **对象池**：`IndicatorPool` 管理 Widget 复用

### Slot 结构

```cpp
struct FSlot
{
    ScreenPosition;     // 屏幕位置
    Depth;              // 深度（Z 排序）
    Priority;           // 优先级
    Visibility;         // 可见性
    DirtyFlags;         // 脏标记
};
```

## 交互系统集成

在 `ULyraGameplayAbility_Interact::UpdateInteractions` 中：

```cpp
for (auto& Option : InteractiveOptions)
{
    UIndicatorDescriptor* Indicator = NewObject<UIndicatorDescriptor>();
    Indicator->SetDataObject(InteractableTargetActor);
    Indicator->SetSceneComponent(InteractableTargetActor->GetRootComponent());
    Indicator->SetIndicatorClass(InteractionWidgetClass);
    IndicatorManager->AddIndicator(Indicator);
}
```

- 交互目标自动生成指示器
- 指示器随交互目标移动
