# 队伍系统

## 架构概览

队伍系统管理玩家的队伍归属、队伍数据显示、队伍得分和团队对抗逻辑。核心由接口、数据资产、Info Actor、子系统和创建组件组成。

## 队伍子系统

[LyraTeamSubsystem.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Teams/LyraTeamSubsystem.h)

```cpp
class ULyraTeamSubsystem : public UWorldSubsystem
{
    TMap<int32, FLyraTeamTrackingInfo> TeamMap;

    bool RegisterTeamInfo(ALyraTeamInfoBase* TeamInfo);
    int32 FindTeamFromObject(const UObject* TestObject) const;
    bool ChangeTeamForActor(AActor* ActorToChange, int32 NewTeamIndex);
    ELyraTeamComparison CompareTeams(const UObject* A, const UObject* B) const;
    bool CanCauseDamage(const UObject* Instigator, const UObject* Target, bool bAllowDamageToSelf = true) const;
    ULyraTeamDisplayAsset* GetTeamDisplayAsset(int32 TeamId, int32 ViewerTeamId);
};
```

- `FindTeamFromObject` — 从任意对象查找队伍 ID（接口→Instigator→TeamInfo→PlayerState）
- `ChangeTeamForActor` — 修改 Actor 队伍（通过 PlayerState 修改）
- `CompareTeams` — 比较两个对象的队伍关系
- `CanCauseDamage` — 判断是否允许造成伤害（友伤判定）

## 队伍创建组件

[LyraTeamCreationComponent.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/Lyra-Deep-Dive/code/Lyra/Source/LyraGame/Teams/LyraTeamCreationComponent.h)

```cpp
class ULyraTeamCreationComponent : public UGameStateComponent
{
    void OnExperienceLoaded(const ULyraExperienceDefinition* Experience);
    void ServerCreateTeams();
    void ServerAssignPlayersToTeams();
    void ServerChooseTeamForPlayer(ALyraPlayerState* PS);
    int32 GetLeastPopulatedTeamID() const;
};
```

1. `OnExperienceLoaded` → `ServerCreateTeams`
2. `ServerCreateTeams` — 为每个队伍生成 `ALyraTeamPublicInfo` + `ALyraTeamPrivateInfo`
3. `ServerAssignPlayersToTeams` — 分配玩家到队伍
4. `GetLeastPopulatedTeamID` — 选择人数最少的队伍
